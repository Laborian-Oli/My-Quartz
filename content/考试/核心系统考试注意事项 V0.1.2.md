# 1 客户改名
根据客户名下的账户类型、数量，执行不同操作。
#### ⭐ **名下只有银行卡**
`8613` 直接修改，系统会自动修改卡账户名和客户名称。

#### ⭐ **名下只有存折、存单**
1. `0201` 修改存折、存单户名
2. `0306` 将存折、存单移出客户号
3. `0313` 修改客户号姓名
4. `0305` 客户总账联结，将移出的存折、存单重新关联到客户号

# 2 转账交易，借贷方凭证

*凭证种类告诉系统“这是什么票”，凭证号让系统“销掉这张票”*；借方那格就填客户交来的那张原始凭证，贷方那格一般填柜员自己打印的内部传票。

所以转账时，借贷方的凭证类型不应该填写借贷方的卡 / 存折，可填写的凭证一般有：
1. 转账支票
2. 结算业务申请书

# 3 定期一本通
## 3.1 查询定期一本通
当使用 `7702` 查询定期一本通内的存款时，将会在账户处显示一本通内部账户，而不是一本通账户，查询到的账号不能办理业务。

需要查询定期一本通账号，可以使用 `0310` 客户统计，查找所有账户，再筛选判断。
知道一本通账号后，在 `7705` 一本通分户查询可以查看：
1. 一本通内部账号
2. 序号

当存款移出定期一本通时，*移出的存单账号就是之前的一本通内部账号*。

## 3.2 账户移入移出一本通
序号会改变，（销户等）应该使用新的存款序号。

原本的序号不会消失，但会被识别为“已经移出”，无法操作。

# 4 存款摘要

考试办理存款时，没有明确要求不需要修改摘要，默认*005-储蓄存款*即可。

> [!tip] 对公存款的情况
> 存款摘要不需要编辑，但**款项来源必须填写**。

# 5 贷款质押
账户冻结类型：**质押止付**。

冻结对象：一般是存单，而不能是定期一本通。

> [!tip] 为什么质押止付的对象不能是定期一本通（及其上的某个定期存款）？
> 1. 贷款质押要求**可交付的权利凭证**，一本通不在此类；
> 2. 一本通上的定期存款不是独立的，不能冻结特定资金；
> 3. 一本通账户结构复杂，不利于内控、风控。

身份核查：贷款质押要求客户到场办理，核查客户身份证即可。

# 6 存款使用什么凭证？
只有**一次性存入**的存款，才可以使用存单作为凭证。

| 存款方式 | 形式                   | 常用凭证类型     | 说明                           | 
| -------- | ---------------------- | ---------------- | ------------------------------ |
| 整存整取 | 一次存入，到期取出     | **存单 / 存折**         | 使用记名存单，凭存单支取本息   |
| 零存整取 | 每月固定存入，到期取出 | **存折**         | 连续多次记账，适合用存折       |
| 整存零取 | 一次存入，分期取出本金 | 存折为主（可能） | 取款灵活，通常用存折           |
| 定活两便 | 不定期，自由支取       | **存单**         | 使用存单，支持灵活支取利率调整 |

这里不考虑子账户。

# 7 身份证有效期更新
交易码 `0422` 、 `0307` 都可以修改身份证有效期，但是 `0307` 需要授权，`0422` 不需要。推荐使用 `0422`。

# 8 单位定期存款以及贷款质押
1. 使用转账开户 `1102` 开立单位定期存款，使用凭证为：`50 - 单位开户证实书`
2. 开立单位定期存单 `1412`，使用凭证为：`115 - 单位定期存单`，然后可以办理质押（题目没有要求）
3. 单位定期存单置换回单位开户证实书，交易码 `1413`

# 9 凭证操作
`7206` 客户票据登记簿，可以查看客户持有的票据及其凭证号。
## 9.1 出售与撤销
-  `0504` 凭证出售
-  `0527` 凭证撤销，撤销时，应当将出售的凭证全部撤销（尚未打印的），然后重新出售正确数额。
## 9.2 凭证核销 / 注销
- `0508` 凭证核销，用于柜员尾箱中的未出售重空核销，输入凭证号即可。
- `0507` 凭证注销，用于已经出售给客户的凭证，需要输入客户账号和凭证号。

# 10 其他
1. 定活两便存款取现 / 转活期，直接销户 / 销子账户。
2. 存单留密码，`0202` 修改密码，旧密码不填，设置新密码。
3. 将凭证上交主管，使用 `0523` 凭证上缴，而不是 `0513` 凭证交接。
4. 协定存款协议，题目要求设置有效期，但核心系统没有输入框，不用设置。
5. 子账户存款转活期，使用 `8620` 销子账户，选择转账，无需输入转入账号，系统自动转入 `卡号-101` 账户。
6. 使用 `1301` 支票取现支取备用金时，摘要应该选择**032-其他支出**。
