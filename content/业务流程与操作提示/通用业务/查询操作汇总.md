# 一般查询操作
## 客户信息查询
- ⭐ `7706`，`0308`
	- 名称：*个人客户信息查询*
	- 功能：
		- 证件号、客户号、客户名称互查
		- 两个交易码功能一致
- ⭐ `4711`
	- 名称：*个人客户信息查询*
	- 功能：
		- 需要客户号和证件号
		- 查询**电子银行限额**等
- ⭐ `0311`
	- 名称：*企业客户信息查询*
- ⭐ `1126`
	- 名称：企业联网核查
	- 功能：
		- 查手机号
		- 时间限制：休息日（周末、节假日）只有第一天能用
## 账户分户查询
- ⭐ `0310`
	- 名称：*客户统计*
	- 功能：
		- 使用证件号
		- 查询各类账户数量、基本信息
		- 结果只能打印，不能复制
- ⭐ `8770`
	- 名称：*查询卡所有账户余额*
### 活期分户
- ⭐ `7701`
	- 名称：*一般活期分户信息*
	- 功能：
		- 活期账户，包括结算账户、活期储蓄账户
		- 无论卡或存折
- ⭐ `0425`
	- 名称：*账户合并信息查询*
	- 功能：
		- 客户结算账户，只能通过证件查询

### 定期和其他存款
- ⭐ `7702`
	- 名称：*整整分户查询*
	- 功能：查询整存整取定期存款，个人或者单位
- ⭐ `7703`
	- 名称：*定活两便分户查询*
- ⭐ `7705`
	- 名称：*一本通分户账查询*
	- 功能：
		- 通过户名、一本通账号查询一本通账户，定期或活期
		- 定期一本通，可以查询到每一笔定期的详细信息，包括**序号**
		- 无法使用证件查询
- ⭐ `7709`
	- 名称：*零整分户查询*
- ⭐ `7710`
	- 名称：*协议存款分户查询*
- ⭐ `7711`
	- 名称：*整零分户查询*
- 大额存单
	- ⭐ `7713`，`7714`，`7715`

### 贷款分户
-  ⭐ `7704`
	- 名称：*贷款分户信息*
-  ⭐ `2830`
	- 名称：*消费贷款分户查询*

`1141`：限额查询

### 其他分户
- ⭐ `7712`
	- 名称：*内部账分户查询*
- ⭐ `6004`
	- 名称：*零余额账户查询*
- ⭐ `7106`
	- 名称：*冻结登记簿*
	- 功能：查询已冻结账户

## 明细和流水
- ⭐ `7801`
	- 名称：*当日明细查询*
	- 功能：
		- 如查询维修基金缴费是否到账
- `4501`：往来流水查询
- `4502`：历史往来流水查询
- `8049`：客户交易明细打印

# 非典型查询方式
通过非查询交易填写户名 / 卡号，自动弹出客户信息的查询方式：
1. `6301` 单笔借贷：账户信息，流水
2. `1302` 有折取款：账户信息
