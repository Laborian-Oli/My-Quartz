对公 & 对私

| 交易码 | 功能               |
| ------ | ------------------ |
| `2301` |                    |
| `2303` | 贷款欠息回收       |
| `2316` | 已核销呆账正常回收 | 

# 1 贷款发放
**贷款出账通知书**：
1. 一式两份
2. 背后都要打印身份核验结果

交易码：根据贷款出账通知书“贷款提示”的要求：
1. 无要求的 `2101`
2.  消费贷款发放 `2811` 

需要补打借方凭证

打印凭证选择：通用贷方，不论公私

**发放完毕后**：
一般取现，具体询问客户经理。
# 2 贷款回收

贷款回收流程：
1. 查询预约，联系客户经理核实
2. 先查贷款账号——贷款分户 `7704`
3. 还款
4. `7801` / `0119` 查询流水，`0808` 补打凭证
5. 根据是否还清
	1. 未还清的，`2301` 打印*还款计划*三份，客户、柜员、客户经理
	2. 还清的，`1104` *销户*（贷款销户后无法重打凭证，慎重操作）


> [!tip] 还款提示
> 1. 贷款还款必须**补打**贷方凭证


## 2.1 按揭贷款回收

`2301`：按揭回收
- 提前还款：提前偿还本金，不影响当期还款
- 本利回收：
	- 必须是扣款当天
	- 偿还当期，不再扣款

`2303`：逾期回收

## 2.2 消费贷款回收
消费贷款欠款还款：`2812`

消费贷款提前还款：`2813`

消费贷销户：`2824`，贷款还完时销户

**消费贷款的现金还款：**
`0604` 现金交易
