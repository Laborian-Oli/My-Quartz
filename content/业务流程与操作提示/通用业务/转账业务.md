# 1 行内转账
## 1.1 
1. 单笔：`6302`
2. 多笔：`6307`
	1. 一借多贷或多借一贷
	2. 不可以多借多贷❌

授权：5 万元以内无需授权
## 1.2 转汇
县级社内部转账

`4301`：转汇申请
- 凭证：结算业务申请书
- 受款方科目号：总是 *2441*

`4305`：转汇凭证

`4306`：封锁转汇
# 2 跨行
## 2.1 电汇

1. **自助预填单**：
	1. `10099`：有凭条，打印后手动勾选转账选项
	2. `8485`：没有单子，用身份证号查询
	3. 凭证单子（结算业务申请书）要剪边
	4. 打印后交由客户核对，签字
	5. 提醒手续费
	6. 若果是公对私汇款：
		1. 必须是基本户
		2. 5 万及以上需要付款依据
2. **身份核验**：
	1. 对私：人证核验
	2. 对公：验印
		1. 系统选择类型：信电汇
		2. 结果打印在结算业务申请书第一联背面
3. **查询行号**：
	1. `4511`
	2. `*关键字*` 使用星号模糊查询
	3. 行号规律：
		- 央行：101 开头
		- 中农工建交：依次往后，102 ~ 106
4. **汇兑录入**：（实时到账）
	1. 再次核对账号、户名
	2. 提醒客户核实账号
	3. 提醒手续费
	4. `4101`
	5. 打印用通用凭证 *（无论对公、对私）*
5. **凭证整理**：
	-  $$通用借方 + 结算业务申请书第一联 + 贷方凭条 + 结算业务申请书第二联 + 手续费借方 + 手续费贷方 + 手续费确认单$$

> [!tip] 免手续费：
> 1. 特殊客户，需要内勤负责人 / 分社负责人签字；没有签字的，会被识别为差错
> 2. 贷款资金，一律免收手续费

## 2.2 交换
使用场景：
- 昆明市内，跨行转账
- 2~3 天到账
- 无手续费

1. 检查汇款凭证，如[[支票业务|支票]] 
2. Excel 手动填写和打印*提出进单*
3. 提款至内部账户，使用 `6302` 或者 `6307`

# 其他
## 账户名填写
**电汇**：即使子账户出款，也应该填写主账户名；
**进账出账单**：两者皆可；
综上，都填写主账户名。
## 凭证日期
结算业务申请书：日期应该为当天
